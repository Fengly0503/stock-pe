## stock-pe
#### 基于NestJs + TypeOrm + Ant Design Pro 搭建股票估值查询系统

主要功能点：
* 自定义返回数据格式
* 自定义异常处理，业务码与http码分离
* TypeOrm基本用法和Entity自动生成
* MySql脚本基础用法
* 请求参数服务端验证
* jwt权限验证
* 用户身份信息注入
* 定时任务同步数据
* Linux下Nginx部署
* Jenkins持续集成/部署
